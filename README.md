# Onsengame [![Gem Version](https://badge.fury.io/rb/onsengame.svg)](http://badge.fury.io/rb/onsengame)

A 2D game framework using Gosu with Ruby.

## Dependencies

* [Gosu](https://www.libgosu.org/)

## Installation

    $ gem install onsengame
